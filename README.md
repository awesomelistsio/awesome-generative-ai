> A curated list of tools, models, datasets, papers, and resources related to generative artificial intelligence.

Generative AI refers to algorithms (like GANs, VAEs, and LLMs) that can generate new content—text, images, music, code, video, and more. This list includes tools for researchers, developers, artists, and entrepreneurs exploring this transformative field.

## Foundations & Concepts

- **[Introduction to Generative AI (Google)](https://cloud.google.com/blog/products/ai-machine-learning/generative-ai-explained)** – Overview of what Generative AI is and how it works.
- **[GANs in 50 Lines of Code](https://towardsdatascience.com/gans-in-50-lines-of-code-torchimpl-708fa47a135e)** – Introductory example for Generative Adversarial Networks.
- **[The Illustrated Transformer](https://jalammar.github.io/illustrated-transformer/)** – Visual explanation of the Transformer model, core to modern generative AI.

## Text Generation

- **[GPT-4](https://openai.com/research/gpt-4)** – OpenAI’s latest large language model for text generation.
- **[Claude](https://www.anthropic.com/index/introducing-claude)** – Constitutional AI-powered chatbot from Anthropic.
- **[Bard](https://bard.google.com/)** – Google’s conversational generative AI model.
- **[LLaMA](https://ai.facebook.com/blog/large-language-model-llama-meta-ai/)** – Meta’s open large language model.
- **[Mistral](https://mistral.ai/news/)** – Lightweight open-source LLMs optimized for performance and scale.

## Image Generation

- **[Stable Diffusion](https://stability.ai/blog/stable-diffusion-public-release)** – Open-source image synthesis model.
- **[DALL·E 3](https://openai.com/dall-e)** – Text-to-image model by OpenAI.
- **[Midjourney](https://www.midjourney.com/)** – High-quality AI art generator from text prompts.
- **[Deep Dream Generator](https://deepdreamgenerator.com/)** – Tool for generating dream-like AI art using convolutional neural nets.

## Music & Audio Generation

- **[Riffusion](https://www.riffusion.com/)** – Real-time music generation using spectrograms and stable diffusion.
- **[Jukebox (OpenAI)](https://openai.com/research/jukebox)** – AI model for generating music with vocals.
- **[Soundraw](https://soundraw.io/)** – AI-powered music generator for creators.
- **[Magenta](https://magenta.tensorflow.org/)** – Research project by Google exploring music and art generation with ML.

## Video Generation

- **[Runway ML](https://runwayml.com/)** – Platform for video creation using generative models like Gen-2.
- **[Pika Labs](https://www.pika.art/)** – AI-generated short-form video from text prompts.
- **[Synthesia](https://www.synthesia.io/)** – Create AI-generated videos with avatars.
- **[Kaiber](https://www.kaiber.ai/)** – Turn audio and prompts into stylized video.

## Code Generation

- **[GitHub Copilot](https://github.com/features/copilot)** – AI pair programmer powered by OpenAI Codex.
- **[Code Llama](https://ai.meta.com/blog/code-llama-large-language-models-coding/)** – Meta’s code generation model.
- **[Replit Ghostwriter](https://replit.com/site/ghostwriter)** – AI coding assistant for Replit users.
- **[Amazon CodeWhisperer](https://aws.amazon.com/codewhisperer/)** – AI coding companion from AWS.

## Multimodal Models

- **[Gemini](https://deepmind.google/technologies/gemini/)** – Google’s multimodal foundation model.
- **[GPT-4V (Vision)](https://openai.com/research/gpt-4v-system-card)** – GPT-4 with visual input capabilities.
- **[CLIP](https://openai.com/research/clip)** – Connects vision (images) and language (text) via contrastive learning.
- **[BLIP](https://github.com/salesforce/BLIP)** – Bootstrapped Language-Image Pretraining from Salesforce.

## Frameworks & Libraries

- **[Hugging Face Transformers](https://github.com/huggingface/transformers)** – State-of-the-art models and tools for text, vision, and audio.
- **[Diffusers](https://github.com/huggingface/diffusers)** – Library for diffusion models from Hugging Face.
- **[LangChain](https://www.langchain.com/)** – Framework for building LLM applications with chaining and memory.
- **[Transformers.js](https://xenova.github.io/transformers.js/)** – Run Hugging Face models in the browser with JavaScript.

## Datasets

- **[LAION-5B](https://laion.ai/blog/laion-5b/)** – Open large-scale dataset for training image-text models.
- **[Common Crawl](https://commoncrawl.org/)** – Petabyte-scale web crawl data for training LLMs.
- **[C4 (Colossal Clean Crawled Corpus)](https://www.tensorflow.org/datasets/catalog/c4)** – Text corpus for language model training.
- **[LibriSpeech](https://www.openslr.org/12/)** – ASR dataset with thousands of hours of English speech.

- **[DeepLearning.AI Courses](https://www.deeplearning.ai/short-courses/)** – AI/ML courses including prompt engineering and diffusion models.
- **[OpenAI Cookbook](https://github.com/openai/openai-cookbook)** – Recipes and examples for using OpenAI’s APIs.
